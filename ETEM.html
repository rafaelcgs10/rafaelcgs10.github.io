1 Cantor

1.1 O Pai da Teoria dos Conjuntos

  1. Georg Cantor

1.2 Cardinalidade de Conjutos

  1. Conjunto de todos os números pares: \(0, 2, 4, 6, 8, 10... \)
  2. Conjunto de todos os números naturais: \(0, 1, 2, 3, 4, 5... \)

    Há mais números naturais do que pares?

1.3 Naturais vs Reais

  1. Todos os números naturais: \(0, 1, 2, 3, 4, 5... \)
  2. Todos os números reais entre 0.0 e 1.0. Segmento de reta continuo de tamanho 1.
  3. Ambos tem infitos números.
  4. Se ambos tem o mesmo número de elementos, então há uma correspodência de 1 para 1.

1.4 Argumento da Diagonalização de Cantor

  1. Por facilidade, vamos representar os números do segumento de reta em base binária.

3 Hilbert

3.1 O Último Matemático Universal

3.2 O Programa de Hilbert

  1. Todos esses paradoxos levaram a matemática para uma crise de fundamentos.
  2. O Programa de Hilbert tinha como objetivo criar uma fundação sólida (consistente) a qual todo a matemática iria se apoaiar:
    1. A matemática deve ser toda escrita de uma linguagem formal, sem ambiguidade.
    2. A formalização precisa ser completa: toda as verdades matemáticas podem ser provadas nesse formalismo.
    3. A formalização precisa ser consistente: não deve ser posssível provar uma contradição nesse formalismo, por exemplo \(0 = 1\).
    4. A formalização precisa ser decidível: há um algoritmo que decide se uma proposição é verdadeira ou falsa.

4 Gödel

4.1 O Melhor amigo de Albert Einstein

4.2 O Teorema da Incompletude de Gödel

  1. Gödel demonstrou que qualquer formalização da matemática com poder suficiente realizar aritmética básica é incompleta.
  2. Gödel criou uma codificação numérica (enumeração) para todas as fórmulas da matemática (em seu sistema).
    1. Uma proposição somente é demonstrável se a sua codificação for divisivel pelas codificações das regras do sistema.
  3. A prova consiste em mostrar a existência de uma proposição \(G\) que é verdade se, e somente se, não for possível prova-la.
  4. \( G \) é uma proposição que fala sobre si mesma, algo similar ao paradoxo do mentiroso:

    Esta afirmação é falsa.

    1. Se a frase é verdadeira, então é falsa.
    2. Se a frase é falsa, então deveria ser verdadeira.
  5. Há uma lacuna entre verdades e provas.
  6. Isso é um resultado muito preocupante, pois diz que há coisas verdadeiras que talvez jamais podemos provar.
  7. Não seria ao menos possível um procedimento que decida todas as proposições que podem ser provadas?

5 Turing

5.1 O Pai da Ciência da Computação

5.2 Indecibilidade do Problema da Decisão (Parada)

  1. Programas, assim como fórmulas matemáticas, podem ser representadas como números. São enumeráveis.
  2. O problema da decisão pede um procedimento (programa) que decida se um dada proposição é verdadeira ou falsa.
  3. Turing criou o conceito de uma máquina abstrata: um humano com papel e caneta fazendo computações/contas/provas.
  4. A ideia é mostrar que não existe máquina abstrata que decida se uma outra máquina para com uma dada entrada.
